# Asset Reconciliation

1. Identify source inventories and their asset kinds.
2. Normalize each item into generic asset fields.
3. Record confirmed links only when the source relationship is explicit or exact.
4. Mark name-similarity matches as candidates.
5. Keep unresolved assets in a missing-link queue.
6. Summarize coverage, risks, and the next provider inventory needed.
7. Keep real inventory data in the local runtime and use only synthetic examples in public files.
